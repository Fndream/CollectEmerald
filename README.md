## 聚集石
由[边界生存](https://github.com/Fndream/BorderSurvivalModpack)聚灵石特性拆分出的独立模组。

## 功能
- 当绿宝石被置于任意物品展示框中时，会聚集周围 33x33x33 范围内的物品到其自身位置；
- 当含有绿宝石的展示框被附着在 箱子、木桶、潜影盒 上时，仅会聚集其附着容器中存在的物品类型。被附着在空容器上时，不会聚集任何物品；
- 当一个物品实体周围存在多个“通用聚灵石”时，该物品会被最近的聚灵石聚集。存在多个有效的“容器聚灵石”时，该物品会分裂为若干个物品实体被这些聚灵石平均聚集；
- 若一个物品实体周围同时存在“通用聚灵石”与“容器聚灵石”，该物品会先被前者聚集，再被后者二次聚集；
- 支持 [carpet-tis-addition](https://modrinth.com/mod/carpet-tis-addition) 模组的 LargeBarrel（大木桶）特性。

## 用途
- 使用此特性在游戏初期轻松实现自动物品分类仓库。
- 将此特性应用于各种陷阱的物品收集装置中。
